贪吃蛇小游戏说明文档
作者：荆薇&徐扬

游戏操作方法：
游戏为简单的贪吃蛇小游戏。
通过键盘上下左右键控制蛇的前进方向吃虫子，虫子随机在地图上刷新。
碰到边界或者吃到自己则死亡，无障碍物。
每吃十个虫子为一关，每上升一关，蛇移动休眠时间减10（移动速度增加），一共20关。


游戏设计思路：
一、首先定义了几个常量。
其中包括：
POS_X（界面横坐标）
POS_Y（界面纵坐标）
SIZE_X（界面宽度）
SIZE_Y（界面长度）
STEP（贪吃蛇移动步长）
MOVE_RATE（贪吃蛇移动休眠时间）

二、Music()播放背景音乐，在setUI()方法中，设置了游戏背景及所需组件。
（包括两张背景图片、蛇、虫子、两个按钮，并设置了它们的坐标、尺寸、颜色）。
并对窗口功能进行设定。

三、paintNumber()中，通过num的控制，进行关卡显示的更新。当num增加到20时，游戏通关并结束。

四、setRandomWorm()中，设置了虫子颜色（深蓝色）及尺寸（设为和蛇的步长一样）、让虫子的坐标随机刷新。

五、keyPressed()对上下左右按键进行读取，并给moveDir变量赋值。
（上键对应3，下键对应4，左键对应2，右键对应1）。

六、moveAuto()方法中对蛇的移动进行设置，添加新线程，来读取键盘输入。
若蛇没有死亡，则让贪吃蛇根据按键（即moveDir变量）进行上下左右的移动。

七、moveSnake()方法中，通过坐标判断蛇是否撞墙死亡，通过循环判断蛇身上每一个像素是否撞到头部死亡。
若未死亡且吃到了虫子，则蛇身长度加一。
每吃10个虫子上升一个关卡，移动速度加快，最大到20关。

八、actionPerformed()方法中通过监听事件，来进行游戏的开始或重载。
